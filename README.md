# Rainbow
Animated rainbow text generator



#### rainbow.object.js

``` 
 Rainbow.defaults.selector = '.text';
 Rainbow.defaults.speed = 100;
 Rainbow.start();
 ```

 
 

#### rainbow.class.js

```
 var rainbow = new Rainbow;
 rainbow.setup({
   selector: '.text'
 });
 rainbow.start();
 ```
